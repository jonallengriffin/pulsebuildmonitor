Get Latest Tinderbox Builds

This is a simple REST API which you can use to retrieve the latest tinderbox builds available on the mozilla FTP site. The return data is a JSON object with the following structure: 
{
  tree: {
    branch: {
      platform: {
        buildtype: buildurl
      }
    }
  }
}
  • tree: currently only mozilla-central is returned
  • branch: either mozilla-central or mozilla-central-mobile-browser
  • platform: e.g., win32, linux, linux64, etc
  • buildtype: either opt or debug
So, if you knew you wanted the url of the latest win32 opt build on mozilla-central, you could retrieve the JSON and do something like: 
  url = data['mozilla-central']['mozilla-central']['win32']['opt']
